Qu'est ce que le développement web?

Le développement Web est un métier associé au développement de sites Web/Application web destinés à être
hébergés via un intranet ou Internet.

Le développeur web

En tant que développeur web, nous devons être capable de savoir mettre en oeuvre la partie front d'un site web, c'est-à-dire sa mise en forme (le design, les animations et autres évènements), ainsi que la partie back, qui consiste à gérer les évènements interne côté serveur (comme une inscription par exemple).

Pour pouvoir fonctionner, une application ou site Web à besoin d’une partie back-end et d’une partie front-end.

Front-end:

Le front-end ou frontend ou front, est TOUT ce que l’utilisateur final peut voir à partir de son ordinateur / smartphone.
Il existe des développeurs qui ne s’occupent que de la partie front-end, ce sont tout simplement des développeurs front-end Les langages les plus connus permettant de développer un site / une appli côté front sont:

  • Le HTML (HyperText Markup Language)
  • Le CSS (Cascading Style Sheets)
  • Le JavaScript

Back-end:

Le back-end ou backend ou back, est TOUT ce que l’utilisateur final ne peut pas voir à partir de son ordinateur / smartphone
. Il existe des développeurs qui ne s’occupent que de la partie back-end, ce sont tout simplement des développeurs back-end Les langages populaires permettant de développer un site côté back sont:

  • Le PHP
  • Le Python
  • Le Java

Fullstack:

Considérés comme les ninjas du développement et capables de prendre en charge un projet de A à Z,
un développeur fullstack est un développeur capable de prendre en charge à la fois la partie front-end et la partie back-end d’une application ou d’un site Web.
